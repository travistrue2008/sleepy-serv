# `sleepy-server`

Mono-repo for `sleepy-serv` server and `sleepy-socket` client packages.

## Important Notes

- This package requires [`bun.sh`](https://bun.sh) instead of NodeJS.
- [`codegraph`](https://github.com/colbymchenry/codegraph) must be installed globally via bun: `bun add -g @colbymchenry/codegraph`
